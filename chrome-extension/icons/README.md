# Extension Icons

You need to add the following icon files to this folder:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can use the Seeksy logo or create simple "S" icons with the brand colors:
- Background: #0a1929 or gradient to #2C6BED
- Text/Icon: White
